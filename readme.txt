Dieses Plugin erlaubt die einfache Integration von Amazon Affialte Links, sogenenannte Produkt Platzierungen mit einfachen Links und Grafiken.

== Description ==

Mit diesem Plugin kannst du Artikel von Amazon.de schnell und einfach in Wordpress Artikel integrieren. 
Mittels eines simplen Iframes werden Artikelbilder sowie der Passende Affiliate Link eingebunden.
Über die Einstellungen werden die nötigen Informationen gesammelt, so das man für die aktive Bearbeitung lediglich die Product ID benötigt.

Ein weiteres Feature ist die Ausrichtung, der sogenannte Textfluss.
Dies ermöglicht auch das aneinanderreihen von Artikel Bildern, das links oder rechtsbündig setzen.

== Installation ==

1. Hier das Amazon Wordpress Plugin herunterladen. 
2. Das so erhaltene Zip-Archiv lokal entpacken und die enthaltene Datei: amazon-einzeltitellinks.php in den Wordpress Plugin Ordner hochladen. 
3. Das Plugin aktivieren und unter Einstellungen deine Amazon Partner_ID eingeben und wenn gewünscht konfigurieren.
